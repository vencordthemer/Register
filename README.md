# Register
a offline school register
> NOT INTENDED FOR A SCHOOL EVIRRONMENT

---
## Instructions

Download the release.zip from the releases section and run the Admins.exe file, a folder should appear, go into that folder and add this to the admins.json file:
```
[
  {
    "name": "Default",
    "password": "1234"
  }
]

```
> You can Change the username a password to anything you want.

After this, reopen the Admins.exe and login with your account.
### Making a teacher account
to make a teacher account go into the admin panel and enter in 1 at the choice, this should show up:


<img width="282" height="179" alt="image" src="https://github.com/user-attachments/assets/1c206428-27aa-4fc1-a405-8418dcf4595f" />
Then you can enter a user and password

### Making a class
To use this you need to make a class to do that select 3 in the admin panel, this should show up:

<img width="297" height="225" alt="image" src="https://github.com/user-attachments/assets/981b9047-203f-4933-90e1-d8c6f46bae2f" />

then type in a name and select your teacher using the dropdown

### Making students and adding them to classes

To make a student select option 2 in the admin panel and this should show up:

<img width="310" height="199" alt="image" src="https://github.com/user-attachments/assets/f042fef5-ef48-4768-85eb-02e8434fc33f" />

Enter a name and year group, now we need to add them to a class to do that select option 4 in the admin panel and this would show up:


<img width="364" height="291" alt="image" src="https://github.com/user-attachments/assets/17cb1f9d-163f-4ea4-b5cf-0dca9cd6334a" />


Select the student and class they should be in

### Teachers using the register

For a teacher to use it open the teachers.exe file and enter your username, password and click login then yout classes will show up:

<img width="220" height="256" alt="image" src="https://github.com/user-attachments/assets/1f1cf808-4cc0-45cc-8c5e-bb5895d0f84c" />

Select your class and use the dropdown to select p (present) a (absent ) and l (late) then save it.


## Features

Adding absenses to a student on all their classes before registers (admin panel)

A chat with teachers  (Admin panel and teacher client)
